The only plugin you need to sell your tickets online with Weezevent.

== Description ==

This plugin was built by [Weezevent](https://weezevent.com?utm_source=wordpress.org&utm_medium=link&utm_campaign=readme) to allow organizers to sell tickets on their website. Using this plugin, you only have to copy/paste the embed code available on your Weezevent dashboard and start selling tickets to your website right away.

== Installation ==

Weezevent plugin can be installed like any other regular WordPress plugin either by uploading the zip file or install it straight from the plugin repository inside the WordPress admin. Once installed, click 'Activate Plugin' to finish the installation.

You can now use Weezevent plugin on your site:
* Go to Pages or Posts
* Choose 'Weezevent Ticketing'

**External service**

This plugin uses Weezevent native code to embed our ticketing widget into your website. This plugin don't have to be linked to your Weezevent Account but you will need one to get the code of your widget.
You can see our terms of use : [here](https://weezevent.com/fr/cgus/?utm_source=wordpress.org&utm_medium=link&utm_campaign=readme) 

**Weezevent Plugin**

After plugin installation, open your WordPress admin panel and do the following:

* Open your WordPress admin panel;
* Select Weezevent on the left panel;
* Copy/Paste the code from Weezevent dashboard;

More information and tutorial videos on [how to use Weezevent plugin](https://https://aide.weezevent.com/?utm_source=wordpress.org&utm_medium=link&utm_campaign=readme).
